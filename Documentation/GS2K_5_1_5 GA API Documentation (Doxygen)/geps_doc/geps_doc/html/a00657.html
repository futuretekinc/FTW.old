Get a bitfield.

This macro extracts a bitfield from a given variable. The returned value is normalized ie., it will be available starting from bit position zero.

Parameters:
var- IN variable from which bitfield has to be extracted.
fieldName- IN name of the bitfield defined earlier using CREATE_BITFIELD.

Get a bitfield.

This macro creates a temporary variable with the bitfield set to given value.

Parameters:
fieldName- IN name of the bitfield defined earlier using CREATE_BITFIELD.
bitFieldValue- IN value to set for bitfield.
Note:
Multiple MAKE_BITFIELDs can be ORed together to form a variable will various bitfields within a 32-bit variable set to different values.
