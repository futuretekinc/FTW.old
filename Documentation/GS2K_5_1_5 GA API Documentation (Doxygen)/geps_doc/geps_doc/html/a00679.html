Decode API for XML module.

This function decode the xml file,extract the data and fill that on user structure.

Parameters:
ParentStart- IN The starting address of the xml file.
ParentEnd- IN The Ending address of the xml file.
XML_Schema- IN Xml schema.
Return_Struct- IN The user structure buffer to be filled with the data.
Return values:
GSN_SUCCESSOperation successful.
GSN_FAILUREOperation failed.

None.

GSN_STATUS_T GsnXml_NodeGet ( GSN_XML_NODE_T *  ChildNode,
UINT8 *  ParentNodeStart,
UINT8 *  ParentNodeEnd 
)

Node get API.

This function gets the node information from the xml file. It fills the pCNode if it finds the tag. The tag name is taken from the pName element in the pCNode.

Parameters:
ChildNode- IN/OUT Child node structure pointer.
ParentNodeStart- IN The starting address of the xml file.
ParentNodeEnd- IN The Ending address of the xml file.
Return values:
GSN_SUCCESSOperation successful.
GSN_FAILUREOperation failed.
Note:
None.
