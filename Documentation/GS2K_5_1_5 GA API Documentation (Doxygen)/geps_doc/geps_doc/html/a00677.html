Detailed Description

Controls for 802.11 operations.

This group contains public definitions for the Wireless Device Driver (WDD). Functions are provided to control 802.11 scanning, connect (associate), disconnect, and transmission and reception of data. IOCTL functions are provided for modifying some WLAN parameters.
