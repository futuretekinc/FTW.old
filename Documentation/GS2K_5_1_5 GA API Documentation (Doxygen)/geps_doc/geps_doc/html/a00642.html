This group contains 802.11 MAC-level definitions.

This group contains macro definitions for authentication, physical-layer properties, encryption, and various MAC address filtering and comparison operations. Note there are a number of macro sequences where only the first element is included in the documentation for simplicity; click on the source code definition link to see all the options for the given macro type.
