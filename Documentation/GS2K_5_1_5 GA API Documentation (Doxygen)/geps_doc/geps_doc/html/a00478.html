typedef struct GSN_COAP_CONN_PARAMS GSN_COAP_CONN_PARAMS_T

Coap request parameters.

This structure defines some of the parameters used for sending the Coap request to the server. This structure needs to be filled and passed to GsnCoap_Send_And_Recieve()
