Detailed Description

This group of API functions implements a generic message handler, by creating a thread and mailbox. The entry function waits on the mailbox and calls the registered callback function.

Create Message Handler. This function is used to create a message handler.

Parameters:
pCtx- IN Pointer to the memory to be used by the message handler.
pCfg- IN Configuration details for the message handler.
pMsg- IN Pointer to the memory in SRAM to read the message from the mailbox. The size of the memory should be same as the size of message stored in mailbox.
handlerCb- IN Handler function to be called to process the message.
pHandlerCtx- IN Handler context to be passed to process the message.
