Detailed Description

The Modules group includes functions and related structures and entities that provide the following services:

  • functions that handle bootup and exceptions;
  • functions that support firmware updates;
  • access to the SOC's non-volatile flash and RAM;
  • access to timers;
  • watchdog controls;
  • power management;
  • die temperature measurement;
  • wrappers on network socket services (TCP/IP).
