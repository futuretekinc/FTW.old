Function to handle scan request.

This API is called to scan ZigBee-IP netwrok.

Parameters:
pZipCtx- IN Pointer to ZigBee-IP context.
channelMask- IN List of channel to scan. The list is bitmap as per the standard 802.15.4-2006 specification.
scanDuration- IN scan duration. The scan duration is as per the standard 802.15.4-2006 specification.
scanBuf- IN Pointer to the buffer where the scan result will be stored.
numOfScanBuf- IN Number of scan result can be stored inteh buffer pointed by scanBuf.
