PUBLIC PL722X_STATUS PL722X_GetOCPParams ( PL722X_OCP_PARAM_T *  param)

Get OCP parameters required for setting loading current and loading time.

Parameters:
param-IN pointer where OCP params read needs to be copied
Note:
please find the formula for computing loading current and time loading curent: STPUTH rule = {{(IA_rms - IA_offset ) / (IA_gain / 2^20 ) } ^2 } * (SampleCnt_1) loading time: STTH rule = {{(IA_rms - IA_offset ) / (IA_gain / 2^20 ) } ^2 } *( SampleCnt_ADC_Clock) *( T_sc)

so parameters needed are -IA_OFFSET (0x3841 - 0x3845) - 5 Bytes -IA_GAIN (0x3822 - 0x3823) - 2 Bytes -SAMPLE_CNT_1 (0x381A - 0x381B) - 2 Bytes -SampleCnt_ADC_Clock = 4000000/1024 = 3906 (d)
