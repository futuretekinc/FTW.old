00237 /*
00238 How to enable OCP (over current protection:
00239 -------------------------------------
00240 ->Read OCP parameters (PL722X_GetOCPParams)
00241 ->Compute corresponding values for  loading time and loading current
00242 ->Configure loading current (PL722X_SetLoadingCurrent)
00243 ->Configure loading time (PL722X_SetLoadingTime)
00244 ->Enable OCP function (PL722X_EnableOCP)
00245 
00246 when over current happens, PL7223 automatically switches off the relay (load) and set the status
00247 register "OC_Flag (0x3024)"
00248 OC_Flag = 0, Normal Condition
00249 OC_Flag = 1, Over-Current Occurred
00250 
00251 
00252 MCU shall read "OC_Flag (0x3024)" to understand the occurance periodically.
00253 If OCP occur, the status flag will remain set until MCU clears it up. So the operation may look like
00254  ->mcu read OC_FLAG (PL722X_GetOCPStatus)
00255  ->reenable the OCP (PL722X_EnableOCP)
00256 
00257 Note: If OCP happened, MCU will not be able to control the load through pl7223 until 
00258          status flag is cleared
00259 */
