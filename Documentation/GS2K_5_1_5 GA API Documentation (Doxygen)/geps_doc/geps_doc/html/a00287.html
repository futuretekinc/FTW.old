Detailed Description

Security EAP FAST Configuration.

This structure defines the EAP-FAST configuration used for configuring the WDD for connecting to a WLAN with EAP-FAST key mgmt.

See also:
GSN_WDD_SECURITY_CONFIG_T and GSN_WDD_WLAN_CONFIG_T.
