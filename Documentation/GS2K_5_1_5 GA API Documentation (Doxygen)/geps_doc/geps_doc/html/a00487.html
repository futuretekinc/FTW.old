)

DTLS Receive API.

This function Processes the data received from the socket descriptor given in GsnDtls_Open API. If received data contains user data, it updates the pInDataLen.

Parameters:
dtlsConn- IN Context of DTLS module.
pInData- IN Pointer to input data.
pInDataLen- IN/OUT Pointer containing the input data length. This pointer is updated to the length of the user data receieved else to 0.
recvFlag- IN flag to use for socket receive API call.
pRecvAddr- OUT Pointer to store the address of the Source device.
pAddrLen- IN/OUT Size of the recvAddr.
Return values:
GSN_SUCCESSif Operation successful, else GSN_FAILURE.
