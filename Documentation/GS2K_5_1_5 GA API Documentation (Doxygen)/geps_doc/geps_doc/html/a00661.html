Detailed Description

DHCP Server API's.

This file contains public definitions and API's for Dhcp Server.


This file contains public definitions and API's for Dhcp Server. DHCP server is started when node acting as a Limited AP. Users can associate to the node and configure it. The API GsnDhcpSrvr_Start starts the DHCP server and API GsnDhcpSrvr_Stop stops DHCP server. When a user/STA disassociates from limited AP its entry can be removed using API GsnDhcpSrvr_ClientDelete.
