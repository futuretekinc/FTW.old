\subsection{Detailed Description}
The Drivers service group provides API's for managing external and on-\/chip peripheral interfaces:
\begin{DoxyItemize}
\item GPIO
\item Flash memory
\item RTC
\item PWM
\item ADC
\item I2C
\item I2S
\item UART
\item Timer
\item SPI
\item SDIO 
\end{DoxyItemize}
